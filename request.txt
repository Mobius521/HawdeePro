智能化在线教学平台功能规整（登录前门户 + 管理员管控）

一、登录前公共门户（全角色共享）
模块	具体功能
平台展示	- 宣传在线教学、直播课、资源管理等核心功能- 展示热门课程 / 直播预告- 介绍平台服务优势
身份引导	- 提供登录入口- 新用户注册入口（关联角色选择）- 展示各角色权限说明
公共信息	- 滚动展示系统公告- 发布近期教学活动通知- 提供平台使用指南 FAQ
二、教师端功能模块
模块	具体功能
系统基础	- 个人信息管理（查看 / 修改）- 信息中心（接收 / 发送 / 回复教学信息）
课程管理	- 查看课表- 创建 / 删除 / 修改课程
课程资源管理	- 展示 / 上传 / 下载 / 删除资源- 资源分类 / 检索- 分享资源给其他教师- 协作编辑资源
试题试卷管理	- 展示 / 下载 / 上传 / 修改 / 删除试题试卷
作业管理	- 发布作业（含限时测试、自动评分）- 批改作业- 查看作业记录 / 详情 / 统计结果
直播课管理	- 创建直播课- 屏幕共享 / 互动问答- 录制 / 剪辑 / 保存课程视频- 投票互动教学
成绩管理	- 展示 / 修改 / 录入成绩- 统计分析成绩- 导出成绩报表
学情分析	- 展示学生学习记录 / 作业情况 / 成绩- 自动生成个性化建议- 修改建议
日志操作	- 查看个人教学操作日志- 统计日志数据
三、助教端功能模块
模块	具体功能
系统基础	- 个人信息管理- 信息中心（接收教师任务）
课程辅助	- 查看课表- 协助创建 / 修改课程资料- 上传 / 下载资源（受限权限）
直播课协助	- 管理直播互动问答- 剪辑 / 保存课程视频
作业辅助	- 查看作业记录 / 详情- 协助批改作业- 查看统计结果
成绩辅助	- 录入成绩- 查看成绩统计- 导出报表（授权课程）
学情跟进	- 查看学生学习数据- 协助生成个性化建议
四、系统管理员端功能模块
模块	具体功能
系统基础	- 个人信息管理- 信息中心（全局消息发布）
网站门户管理	- 发布 / 修改 / 删除系统公告- 配置首页轮播图 / 热门课程- 管理角色登录入口（增删角色选项）- 调整门户界面布局（导航栏 / 数据卡片）
用户管理	- 创建 / 删除教师 / 助教账号- 配置账号权限- 查看用户操作日志
课程监管	- 审核课程创建 / 删除- 统计全局课程数据- 监控课程合规性
资源监管	- 监控资源存储状态- 删除违规资源- 配置资源分类规则
系统运维	- 查看全量操作日志- 监控系统性能- 执行数据备份 / 恢复
五、功能权限逻辑说明
1.	公共门户与登录流程
o	未登录用户仅可浏览公共门户内容，通过门户内的登录 / 注册入口进入系统；
o	登录后根据角色跳转至对应工作台（教师端 / 助教端 / 管理员端）。
2.	管理员对门户的管控
o	系统管理员可完全控制公共门户的展示内容（公告、宣传图等）和角色入口配置；
o	教师 / 助教无门户管理权限，仅能使用个人工作台功能。
3.	跨角色功能关联
o	教师发布作业 / 直播课→助教可协助管理→学生（隐含角色）接收任务；
o	管理员审核课程→教师方可发布课程；
o	管理员配置门户公告→所有用户登录前可见。

六、个人任务
教师端
•	课程管理：查看课表、创建课程、删除课程、修改课程。
•	课程资源管理：展示所有资源、上传资源、下载资源、删除资源、资源分类、资源检索、资源分享给其他老师、协作编辑资源。
•	日志操作管理：查看日志、统计日志。
•	分析学生学情：展示学生学习记录、展示学生作业情况与成绩、自动生成个性化建议、修改个性化建议。
•	作业管理：展示作业记录、查看作业详情、查看统计结果、发布作业、批改作业、限时测试、自动评分。
助教端
•	课程管理：查看课表、协助创建 / 修改课程资料（受限权限）。
•	课程资源管理：上传 / 下载资源（受限权限）。
•	日志操作管理：无明确对应功能。
•	分析学生学情：查看学生学习数据、协助生成个性化建议。
•	作业管理：查看作业记录、查看作业详情、查看统计结果、协助批改作业。
系统管理员端
•	课程管理：审核课程创建 / 删除、统计全局课程数据、监控课程合规性。
•	课程资源管理：监控资源存储状态、删除违规资源、配置资源分类规则。
•	日志操作管理：查看全量操作日志、统计日志数据。
•	分析学生学情：无明确对应功能。
•	作业管理：无明确对应功能。
公共门户与跨角色功能
•	登录 / 注册：未登录用户通过 “登录（/login）” 入口进入系统，登录后按角色跳转至对应工作台。注册界面在登录界面跳转。
•	网站门户（首页）：系统管理员可管控首页轮播图、热门课程展示、公告发布等内容，教师 / 助教无门户管理权限。
